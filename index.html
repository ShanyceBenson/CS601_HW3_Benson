My Projects

Validation Form

This project is a simple subscription form that collects a user's first name, last name, email, package selection, and subscription confirmation. It includes client-side validation using JavaScript to ensure proper input formats and required field completion, excluding the optional email field. If all inputs are valid, a personalized summary message is displayed, and the form is hidden using DOM manipulation. The form's validation logic uses regular expressions and the forEach() high-order function for cleaner, modular code.

Check it out on Github

Drag and Drop Game

This project is a simple web-based drag-and-drop game where users categorize a list of movies and sitcoms. The data is loaded dynamically from a package.json file and randomly shuffled each time the page loads. Users drag each item into the appropriate drop zone based on its category. The application is built with HTML, CSS, and JavaScript and demonstrates event handling, DOM manipulation, and basic interactive UI design.

Check it out on Github

MyPortfolio

This project is a responsive one-page portfolio built using only HTML5 and CSS, styled in a custom dark mode theme. It includes a header and footer designed with Flexbox for proper alignment, a CSS Grid-based project gallery, and a sidebar with personal information. Interactivity is added through pseudo-classes like :hover, :focus, and :nth-child(n), and enhanced visuals are created using a ::before pseudo-element. The layout also demonstrates advanced CSS skills like combinators, attribute selectors, and ensures the footer stays at the bottom of the page regardless of content height.

Check it out on Github
